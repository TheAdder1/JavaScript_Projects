Fizz Buzz

Receive an integer where you must loop through all preceding numbers and output them as an array that display 'Fizz' if divisible by 3, 'Buzz' if divisibile by 5, 'FizzBuzz' if divisible by 3 and 5, and the number itself for all other scenarios.
Input: n=15
Output: [1, 2, Fizz, 4, Buzz, Fizz, 7, 8, Fizz, Buzz, 11, Fizz, 13, 14, FizzBuzz]

Input: x=2 y=3
Output: [6,12,18,24,30]

Roman Numeral

Convert a string roman numeral 'roman' to an integer
I = 1
V = 5
X = 10
L = 50
C = 100
D = 500
M = 1000
I can be placed before V(5) to make 4 and before X(10) to make 9
X can be placed before L(50) to make 40 and before C(100) to make 90
C can be placed before D(500) to make 400 and before M(1000) to make 900
Input: roman="XLIX"
Output: 49

Input: haystack="basketball" needle="sk"
Output: 2
